This simple database stores First Names, Last Names, and Phone Numbers onto your web browser's local storage.

Features:

  • The ONLY required field is First Name
  • Last Name is optional
  • Phone is optional but must be a number in 10 digit format with hypens, spaces, or simply 10 digits
  • Database entries are stored in alphabetical order by First Name
  • You may edit any field
  • You may delete any database entry
  • If you close your browser and re-open it, the database entries are still there.
  • ENJOY!
